ENTERPRISE RAG CHATBOT

This project implements an Enterprise Chatbot tailored for Payments and Settlement Systems, featuring a user interface powered by Streamlit. The chatbot incorporates cutting-edge NLP techniques facilitated by LangChain. Key components include an Embeddings Model utilizing HuggingFace's all-MiniLM-L6-v2 for generating precise sentence embeddings. For document retrieval based on similarity metrics, the project employs FAISS as a vector store. Additionally, the chatbot leverages the llama-2-7b-chat.ggmlv3 model for advanced conversational capabilities, ensuring interactions within the domain of Payments and Settlement Systems.

EDA REPORT PYTHON PACKAGE

EDAExcelReport is a Python package designed to automate the generation of detailed exploratory data analysis (EDA) reports for datasets with binary target variables. It calculates the frequency and distribution of feature values, computes metrics like target rate and lift, and handles both numeric and categorical data seamlessly. The reports generated are in Excel format, featuring well-formatted tables with conditional formatting for improved readability, helping analysts and data scientists understand feature distributions and relationships with the target variable effectively.
